Basic Git Command Instructions

(ls and pwd can be done at any time)
Step # Command to type What it means
#1 cd Documents First lets get us into where we will be saving all our code
#2 git clone https://github.com/beedevservices/beedev-helper.git This will bring the repository down to your pc in the current folder
#3 cd beedev-helper Now we need to move into that new repository folder
#3a For this step we are going to assume you have now edited the index.html file
#4 git status This tells us what files are new or modified in this case just modified
#5 git add . This simply adds all the files that have changed into a "box"
#6 git commit -m 'updated index page' This seals the "box" and puts a message letting others know what happened
#7 git push This ships off the "box" back up to git hub
#8 All commands: git clone, git status, git add, git commit, git push
