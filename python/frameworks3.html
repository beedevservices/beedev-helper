Getting started with Flask

Installation and project setup

Create a directory for your project and move into it 

    $ mkdir myproject
    $ cd myproject

Create a virtual environment 

    $ python -m venv venv

Activate the environment 

    $ source venv/bin/activate

Install Django 

    (venv) $ pip install Django==4.2.8

Create a new project 

    (venv) $ django-admin startproject myproject

This creates the following structure (along with some additional Django files) 

    myproject/
        manage.py
        myproject/
            asgi.py
            __init__.py
            settings.py
            urls.py
            asgi.py
            wsgi.py

Enter the project folder 

    (venv) $ cd myproject

You can verify that everything works by running following command 

    (venv) $ python manage.py runserver

Open localhost:8000 in your web browser and you'll see the success screen. You can stop the server with <control-c>.

Your first app

To create your first app, make sure you're in the folder with manage.py then type this command in the terminal 

    (venv) $ python manage.py startapp firstapp

Add your first view by modifying the views.py file in the firstapp folder as follows: 

    from django.http import HttpResponse


    def index(request):
        return HttpResponse("Hello, world!")

Create a file named urls.py in the firstapp folder and add the following code: 

    from django.urls import path
    from . import views

    urlpatterns = [
        path('', views.index),
    ]

In the myproject folder, modify the urls.py file that was automatically created. This connects the urlpattern we defined in our app to the project urls. 

    from django.contrib import admin
    from django.urls import include, path

    urlpatterns = [
        path('admin/', admin.site.urls),
        path("", include("firstapp.urls"))
    ]

Restart the server if you had stopped it before and navigate to localhost:8000 again. You should now see the "Hello, World!" text we added in our view instead of the success page. Congratulations!
